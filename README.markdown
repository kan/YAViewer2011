# YAViewer2011

## これは何?

YAPC::Asia 2011 (http://yapcasia.org/2011/) のセッション一覧を参照し、自分の見たいトークを管理できるAndroid用ビューワです。
大体2010年版で出来たことは同様に出来るようにしたいと思っています。

- 日時/会場別のトーク一覧
- トーク詳細
- 見たいトークのクリップ/クリップ一覧

追加して、今年は(余裕があれば)以下の機能も実装予定です

- 同日の全会場トークオーバービュー
- 検索
- 見たいトークの5分前通知
